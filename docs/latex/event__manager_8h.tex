\doxysubsection{Detailed Description}
\doxylink{structEventManager}{Event\+Manager} module for managing event listeners in a flexible and efficient manner. 

This module provides functionality to create, manage, and destroy an \doxylink{structEventManager}{Event\+Manager}, which can handle multiple lists of event listeners. It allows adding, removing, and clearing listeners for specific events or all events.

This function deallocates all memory associated with the \doxylink{structEventManager}{Event\+Manager}, including the memory for each event listener and the array of listener lists. It also clears the memory of each node before freeing it to ensure no garbage values are left behind.
