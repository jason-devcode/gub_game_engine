\doxysubsection{Detailed Description}
Structure that defines the properties of the graphic environment. 

This structure holds the configuration and state of the graphic environment used by the game engine.

This structure holds information about the graphic environment used by the game engine. In SDL 1.\+2, it includes a pointer to the screen surface.
